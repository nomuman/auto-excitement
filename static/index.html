軸の定義（経験則プロキシ + 純データ駆動の latent 空間）

1次元の「興奮 ↔ 退屈」では脳状態を捉えきれないため、4 つの仮置き軸 + PCA 上位 3 成分を返します。軸の名前は事後解釈です:

  • Excitement = (z(Visual+SomatoMotor+DorsalAttn+VentralAttn) − z(DefaultMode))/2 — 感覚・注意の関与
  • Valence = z(Limbic) — 感情価のプロキシ（OFC/vmPFC を含む辺縁系）
  • Cognitive Load = z(mean(FrontoParietal, DorsalAttn)) — 制御要求／作業記憶
  • Novelty = z(VentralAttn − mean(全 Yeo7)) — Salience が baseline を超える量（≈予測誤差）
  • PC1–3 — 20484 頂点空間の PCA 上位 3 成分（軸の意味は post-hoc）。PCA の累積寄与は data.pca_var で取得
